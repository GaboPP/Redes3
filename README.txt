Para la ejecución de la tarea se debe abrir un terminal en la carpeta actual y escribir el comando

	make compile

Para la compilación del código con maven (ejecutando "install", "compile" y "package").

Luego se debe iniciar el programa con el comando:

	make run
Lo que dara incio al menu de seleccion entre Servidor y Cliente.

Servidor informa puerto que esta disponible para conexion y asi mismo lo hace el cliente para la retransmision

 java 1.8+
